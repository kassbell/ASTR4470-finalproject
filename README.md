# ASTR4470-finalproject - N-Body Gravitational Simulator
This project implements a numerical simulator for gravitational N-body systems.

## Features (planned)
- Pairwise gravitational force calculation
- Multiple integrators (Euler, RK4, Leapfrog)
- Energy and angular momentum diagnostics
- Visualization of particle trajectories

## Structure
- `src/forces.py`: force calculations
- `src/integrators.py`: time integration methods
- `src/simulation.py`: main simulation loop
- `src/analysis.py`: diagnostics
- `run_simulation.py`: driver script
